### Multi-agent systems and belief graphs

The bet: owning the weights turns a model's internal state into a read-and-write control surface, and for an agent fleet writing to a shared belief graph that control beats a stronger sealed API.

- **Agents collaborate in latent space**, not lossy text — up to ~4× faster on 70–84% fewer tokens, and no text round-trip for drift to compound through.
- **Probes read internal confidence** to gate every memory write and weigh consensus, so the belief graph stays clean instead of poisoning on confident-sounding writes.
- **Every belief is grounded in the model's state at assert-time**, so the whole run replays and audits, bit for bit. Provable, not plausible.
